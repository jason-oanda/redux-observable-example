## Quick Start
```bash
#clone the repo
git clone https://github.com/monad98/redux-observable-example.git

#change directory to repo
cd redux-observable-example

# install dependencies
npm install

#start
npm run serve
//and 
npm run dev
```

## Packages used
- React
- redux
- redux-observable
- react-router 4
- react-router-redux
